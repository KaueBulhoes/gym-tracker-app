# 2026-04-09 — Tela de criação de plano de treino e refatoração de navegação

## O que foi feito

- Criada tela `AddWorkoutPlanScreen` com 3 modos de divisão: ABCDE, Seg a Sex e Personalizável
- Criado componente `DayChipSelector` — chips togláveis com suporte a itens extras auto-selecionados
- Criado componente `CustomDayForm` — form sequencial com rename inline e delete por item
- Criado `AppNavigator` (Stack) para substituir o `HomeScreen` direto no `RootNavigator`
- Adicionados tipos `AppStackParamList`, `HomeScreenProps`, `AddWorkoutPlanScreenProps` em `navigation/types.ts`
- Adicionado FAB centralizado no rodapé da `HomeScreen` para navegar para `AddWorkoutPlan`
- Corrigidas chaves de frontmatter (`agent:` → `mode:`) nos prompts do Copilot

## Decisões tomadas

- Chips extras (F, G... / Sáb, Dom) são removidos da lista ao serem desmarcados, mantendo apenas os itens base
- O botão "+" de novos chips já auto-seleciona o item adicionado (UX mais fluida)
- Navegação migrada de `HomeScreen` direta para `AppNavigator` (Stack) para suportar fluxo de telas futuras

## Problemas encontrados

- `ref` em `styled.TextInput` não é suportado diretamente — resolvido com `React.forwardRef` wrapper
